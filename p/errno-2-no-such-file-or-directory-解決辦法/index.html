[Errno 2] No such file or directory 解決辦法

can't open file 'main.py': [Errno 2] No such file or directory

May 24, 2021

python 環境下使用terminal開啟.py檔案 出現 [Errno 2] No such file or directory

我們在terminal運行 python3 main.py 卻出現錯誤，繁複檢查檔案，檔案確實在專案底下。/

這個問題主要是因為python環境變量設在Python.framework了，最簡單的解決辦法就是使用以下指令

python3 /Users/jameschiang/PycharmProjects/words-arrange/mian.py
# 直接導向檔案位置
